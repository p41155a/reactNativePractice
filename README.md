## React Native를 이용해 만든 앱등이의 애플 제품 및 애플 스토어 찾기 어플

## 메인 페이지
#### 검색바와 테이블로 이루어져 있는 페이지

## 상세 페이지
#### 이미지와 상세 text 들로 이루어져 있고 뒤로 가기 버튼이 있음

## 탭바
#### 두개의 탭바로 구성

## 맵뷰
#### 애플스토어 두개를 버튼으로 이동이 가능하며 key 값을 통해 페이지가 달라짐
#### 또한, 카카오맵으로 모달뷰로 상세 페이지를 볼 수 있음
